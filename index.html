How to use?

Add the repo to your helm:

helm repo add besmirzanaj https://besmirzanaj.github.io/charts/

Charts

  • ifconfig (v0.1.0) - A Helm chart to deploy echoip with GeoIP databases (GeoLite2)
  • ripe-atlas-probe (v0.3.0) - A Helm chart to deploy a RIPE NCC software Atlas probe with automatic SSH key generation
  • validate-email (v1.2.1) - Email deliverability validation API.

Install Guide

To install a chart from this repo:

helm install my-release besmirzanaj/<chart-name>

To uninstall the chart:

helm delete my-release

For more info on charts, see the main repo at https://github.com/besmirzanaj/charts.
